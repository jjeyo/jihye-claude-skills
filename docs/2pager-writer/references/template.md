## 8. AB Test 진행 여부

**작성 가이드:**
- 신규 프러덕트 기능 관련 AB test 진행 필요여부 검토
- 35점 중 26점 이상인 경우 실험 강제 적용 권고

| 진행 필요 | 진행 불필요 |
|----------|------------|
| ☐ | ☐ |

### 총점 기준 (총합: 35점)
- **26점 이상**: 실험 강제 적용 권고
- **20~25점**: 실험의 타당성 및 우선순위 고려하여 결정
- **19점 이하**: 사후 분석으로 대체

| 평가항목 | 설명 | 스코어 기준 | 산정 점수 |
|---------|------|------------|----------|
| 비즈니스 KPI 영향도 | 핵심 KPI에 얼마나 직접적인 영향을 주는가? | 1: 간접/무관, 3: 경로 중간 요소, 5: 직접 연결 | |
| KPI 변화 예상 폭 | 해당 변경이 KPI에 얼마나 큰 uplift 가능성을 갖는가? | 1: 1% 미만, 3: 3~5%, 5: 10% 이상 | |
| 기대 효과의 불확실성 | 변경 결과의 효과에 대한 예측 가능성 | 1: 매우 명확, 3: 팀 내 의견 갈림, 5: 완전 신규 | |
| 대체 분석 가능성 | 실험 없이도 효과 검증 가능한 대체 방법 존재 여부 | 1: 완전히 대체 가능, 3: 일부 대체 가능, 5: 전혀 불가능 | |
| 의사결정 난이도 | 팀 간 의견 일치/불일치 정도 | 1: 전원 합의, 5: 강한 충돌 | |
| 변경 범위 및 릴리즈 위험도 | 실패 시 부정적 영향 | 1: 무해함, 5: 치명적 | |
| 기회비용 대비 학습 가능성 | 실험을 통해 학습하거나 재사용 가능한 가치 | 1: 이미 알고 있음, 3: 상황 의존적, 5: 전혀 새로운 인사이트 | |
