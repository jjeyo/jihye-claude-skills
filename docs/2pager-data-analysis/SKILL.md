# 2-Pager Data Analysis

2-pager 문서 작성을 위한 데이터 분석 및 레퍼런스 수집 스킬입니다.
이 스킬은 반복 호출하여 추가 데이터 수집 및 인사이트 보강이 가능합니다.

---

## 워크플로우 개요

1. **분석 주제 확인** → 어떤 문제/가설에 대한 데이터가 필요한지 파악
2. **데이터 소스 선택** → 필요한 소스(Wiki, Amplitude, Jira 등) 결정
3. **데이터 수집** → 각 소스에서 관련 데이터 검색 및 추출
4. **인사이트 정리** → 수집된 데이터를 정리하고 핵심 인사이트 도출

---

## Step 1: 분석 주제 확인

사용자에게 다음을 질문합니다:

> **어떤 주제에 대한 데이터를 수집할까요?**
>
> 예시:
> - "발견탭 전환율 현황과 개선 기회"
> - "홈탭 커뮤니티 기능 사용자 반응"
> - "선물하기 기능 활성화 지표"

---

## Step 2: 데이터 소스 선택

분석 주제에 따라 적절한 소스를 선택합니다:

| 소스 | 적합한 데이터 유형 |
|------|-------------------|
| Confluence Wiki | 기존 정책, 히스토리, 기획 문서 |
| Jira | 진행 중인 이슈, 백로그, 실험 결과 |
| Amplitude | 사용자 행동 데이터, 전환율, DAU 등 |
| Google Drive | 경쟁사 분석, 외부 리서치 자료 |

---

## Step 3: 데이터 수집

### 검색 도구

- **Confluence Wiki**: `mcp__atlassian__confluence_search` 또는 CQL 검색
- **Jira**: `mcp__atlassian__jira_search` (JQL 검색)
- **Amplitude**: `mcp__Amplitude__search`, `mcp__Amplitude__query_dataset`, `mcp__Amplitude__query_charts`
- **Google Drive**: `mcp__google-drive__search` (MCP 연결 필요)

### 수집 대상 체크리스트

- [ ] 현재 비즈니스 지표 (GMV, 전환율, DAU 등)
- [ ] 사용자 피드백 및 Pain points
- [ ] 경쟁사 분석 자료
- [ ] 관련 기존 문서 및 정책
- [ ] 실험 결과 및 인사이트
- [ ] VOC 데이터

### 출처 표기 규칙

모든 데이터에 `Source: [링크]` 형태로 출처를 명시합니다.

```
예시:
- 홈탭 DAU: 150만 (Source: [Amplitude 차트](링크))
- 발견탭 전환율: 2.3% (Source: [Weekly Report](링크))
```

---

## Step 4: 인사이트 정리

수집된 데이터를 다음 형식으로 정리합니다:

### 출력 형식

```markdown
## 수집된 데이터 요약

### 핵심 지표
| 지표 | 현재 값 | 비교 기준 | Source |
|------|---------|----------|--------|
| ... | ... | ... | ... |

### 주요 인사이트
1. [인사이트 1] - Source: [링크]
2. [인사이트 2] - Source: [링크]

### 추가 분석 필요 항목
- [ ] ...
- [ ] ...

### 참고 문서
- [문서명](링크)
```

---

## 도구 호출 실패 시 대응

| 상황 | 대응 |
|------|------|
| Confluence/Jira 접근 실패 | 사용자에게 Atlassian 연결 상태 확인 요청 |
| Amplitude 데이터 없음 | 사용자에게 직접 데이터 제공 요청, 또는 "데이터 미확보" 표기 |
| Google Drive 미연결 | MCP 서버 설정 필요 안내, 또는 사용자가 직접 파일 공유 |

---

## 반복 호출 가이드

이 스킬은 반복 호출하여 추가 데이터를 수집할 수 있습니다:

> **추가로 분석할 내용이 있나요?**
> - 다른 지표 확인
> - 다른 소스에서 데이터 수집
> - 특정 기간/세그먼트 심화 분석

모든 분석이 완료되면 `/2pager-writer`를 사용하여 문서 작성을 시작합니다.
